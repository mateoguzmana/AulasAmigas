Pasos para ejecutar el sistema.
1. Ejecutar la API en el puerto 8080 (localhost), esto es PHP, entonces requerimos un servidor local, por ejemplo (xampp, wamp)

- El servicio del Angular hará una petición así "http://localhost:8080/api/controllers/archivo.controller.php"

2. Instalar dependencias en el paquete de Angular 2.
Para instalar estas dependencias es necesario tener instalado npm y nodejs.

Por favor, en el directorio raíz del la carpeta App ejecutar el siguiente comando:

    npm install

Una vez estén instaladas las dependencias, ejecutar el siguiente comando para iniciar el proyecto

    npm start

El proyecto una vez ejecutado estará corriendo en el puerto 3000 (http://localhost:3000)
